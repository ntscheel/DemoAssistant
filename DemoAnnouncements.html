New Feature: Generic Responses

Published 2017-01-17 11:46:53 by Nick Scheel

Hello TD!


We have added a new feature that allows you to easily create and access email templates to copy/paste into tech notes. It can be found on the home page in the center section.


To add a new template, simply click the provided link which will send you to the Generic Response Edit page. From here, there are two tabs. One will allow you to create a new response, and the other will allow you to delete them.


If you run into any problems with this new feature, please let us know at our Feedback Page.

Introducing Notifications!

Published 2016-12-07 14:43:22 by Nick Scheel

Hello Beta testers!


We have begun the implementation of notifications, which can be seen in the "Notifications" tab here in the center of the home page. When you receive a new notification, you will see a number indicating how many are unseen. These unseen notifications will also have a bubble "New" next to them. To mark them as seen, simply click the "Dismiss" link to the right of each notification.


By default, only the 5 most recent notifications will be shown. To see more, click the link labeled "Show More Notifications" at the bottom.


As always, if you run into any problems with this feature, let us know by submitting feedback here!
